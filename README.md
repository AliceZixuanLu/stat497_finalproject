# Project: Kobe Bryant Shot Analysis

## Project Overview

This project involves analyzing a dataset of Kobe Bryant's basketball shots to uncover patterns and insights about his shooting performance. The analysis includes data cleaning, exploratory data analysis (EDA), and visualization of shot locations and success rates.

## Dataset

Data available at:
https://www.kaggle.com/c/kobe-bryant-shot-selection/data

## Install and Run

1. Install the required packages:

   ```bash
   pip install -r requirements.txt
   ```

2. Run the Jupyter notebook:
   ```bash
   jupyter notebook kobe_shoot_analysis_final_1.1.ipynb
   ```
